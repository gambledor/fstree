# fstree
File System Tree print out is my custom tree unix shell command.

```
Usage of fstree:
  -a    Show dot files
  -d    Show directories only
  -e string
        Exclude listed directories.
        Example: foo,bar,baz
  -l int
        Limit the level of recursion depth  (default 4294967295)
  -version
        Show current program version
```
